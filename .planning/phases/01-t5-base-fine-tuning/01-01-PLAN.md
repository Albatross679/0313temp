<objective>
Fine-tune T5-base on NL-to-SQL using a W&B Bayesian hyperparameter sweep, evaluate the best config against T5-small baseline, and generate test outputs if T5-base wins.

Purpose: Find optimal T5-base hyperparameters via sweep rather than guessing a single config. Establish whether the larger model provides a stronger Part 1 baseline than T5-small (dev F1=0.618) before DPO training in Phase 2.
Output: Best T5-base model checkpoint, dev metrics comparison, and (conditionally) test SQL/records files.
</objective>
